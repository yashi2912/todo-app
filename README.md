# 📝 To-Do App

A simple, modern, and responsive To-Do Application built using **HTML, CSS, and JavaScript**.
This app helps users manage daily tasks efficiently with a clean and attractive user interface.

---

## 🚀 Live Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 🎨 Attractive UI with gradient design
* 📱 Responsive layout (works on desktop & mobile)

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & UI Design
* **JavaScript (Vanilla JS)** – Functionality

---

## 📂 Project Structure

```bash
todo-app/
│── index.html
│── style.css
│── script.js
```

---

## ⚙️ How to Run Locally

1. Download or clone this repository:

```bash
git clone https://github.com/your-username/todo-app.git
```

2. Open the project folder

3. Double click on `index.html`
   OR open with Live Server in VS Code

---

## 🎯 How It Works

* User enters a task in the input field
* Clicks the **Add** button
* Task gets added to the list
* Use ✔ button to mark task as done
* Use ✖ button to delete task
